# Posts LinkedIn — Projet CDA (CRM)

---

## VERSION 1 — Sobre & Professionnelle

---

Construire un CRM complet, de la base de données jusqu'à la mise en production — c'est le projet que j'ai réalisé pour valider ma certification **Concepteur Développeur d'Applications**.

L'application permet de gérer des clients, créer des devis et générer des factures. Un outil métier concret, pensé pour être utilisable en conditions réelles.

**Stack technique :**
– Frontend : React.js + Vite
– Backend : Node.js + Express + Prisma
– Base de données : PostgreSQL
– Déploiement : Docker + Caddy + GitHub Actions (CI/CD)

Ce qui m'a le plus appris dans ce projet, c'est la partie que l'on voit le moins : concevoir une architecture cohérente dès le départ, sécuriser les accès, automatiser le déploiement. Autant de compétences qui ne s'improvisent pas.

👉 L'application est en ligne : www.m-atici.fr

Retours et critiques bienvenus — je suis toujours preneur de regards extérieurs.

Un CRM déployé en prod. Développé seul. Dans le cadre de ma certif CDA.
